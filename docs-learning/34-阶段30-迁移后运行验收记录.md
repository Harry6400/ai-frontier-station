## 7. 答辩可以怎么讲

如果老师问“迁移后你怎么保证系统还能跑”，可以这样回答：

> 我没有只看代码能不能编译，而是做了完整冒烟验收。首先验证未登录访问后台接口会返回 401，说明 Spring Security 和 JWT 生效；再用默认管理员账号登录，携带 token 访问 Dashboard、API 设置和内容管理接口。前台门户接口仍然公开访问，说明前后台权限边界没有互相影响。最后我还验证了 GitHub、arXiv、HuggingFace 三类外部数据源接口和后台弹窗入口，确认迁移后的新增能力可以演示。

如果老师问“API Key 安全怎么做”，可以这样回答：

> 后台页面可以输入百炼、MiMo、GitHub Token，但接口不会返回完整明文。后端使用 AES-256-GCM 加密后保存到数据库，只在页面展示掩码后缀和保存位置。开发环境有默认 master key 方便演示，但生产环境必须通过环境变量替换为强密钥。
